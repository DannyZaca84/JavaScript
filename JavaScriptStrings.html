JavaScript String Methods

String methods help to work with strings.

String Methods and Properties

Primitive values, like "Daniel Luna", cannot have properties or methods (because they are not objects). But with JavaScript, methods and properties are also available to primitive values, because JavaScript treats primitive values as objects when executing methods and properties.

String length

THe length property returns the length of a string. Example:

let txt = "abcdefghijklmnopqrstvwxyz"

let length = txt.length;
