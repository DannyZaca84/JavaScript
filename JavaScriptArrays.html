JavaScript Arrays

An array is a special variable, which an hold more than one value. For example:

const cars = ["Toyota","Nissan","Chevy"];


Why Using an Array?

If you have a list of items (a list of car names, for example), storing the cars in single variables could look like this:

let car1 = "Saab";
let car2 = "Volvo";
let car3 = "BMW";

However, what if you want to loop through the cars and find a specific one? And what if you had not 3 vars, but 300?
The solution is an array!
An array can hold many values under a single name, and you can access the values by referring to an index number.

Creating an Array

Using an array literal is the easiest way to create a JavaScript Array.

Syntax:

const array_name = [item1, item2, ...];
NOTE: It is common practice to declare an array with the 'CONST' keyword.

It is also possible to create an array, and then provide the elements later:

const cars = [];
cars[0]= "Mitsubishi";
cars[1]="Volvo";
cars[2]= "BMW";
cars[3] = "Toyota";

Another way to crate an array is by using the key word new:
const cars = new Array("Saab", "Volvo", "BMW");

Accessing Array Elements

To access an array element, simply refer to the INDEX NUMBER, for example:

const cars = ["Toyota","Nissan","Chevy"];
let car = cars[2];

Changing an Array Element

This statement changes the value of the first element in cars:

cars[0] = "changed";

    The Compare Function

    The purpose of the compare function is to define an alternative sort order. The compare functions should return a negative, zero, or positive value, depending on the arguments.

    function(a,b){return a - b}

    When the sort() function compares two values, it sends the values to the compare functio, and sorts the values according to the returned (negative, zero, positive) value.

    If the result is negative a is sorted before b.

    If the result is positive n is sorted before a.

    If the result is 0 no changes are done with the sort order of the two values. Example:

    The compare function compares all the values in the array, two values at time (a, b).
